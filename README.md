# Trabalho IA 2

Este projeto consiste em uma análise exploratória de dados (EDA) detalhada sobre um dataset de jogos de tabuleiro. O objetivo é entender os fatores que influenciam a popularidade e a avaliação dos jogos, além de aplicar técnicas de limpeza, engenharia de atributos e detecção de anomalias.

---

## Visão Geral
O projeto analisa características como tempo de jogo, número de jogadores, mecânicas e editoras para responder perguntas como:
* Jogos mais complexos ou longos têm notas melhores?
* Quais são as franquias de jogos mais dominantes?
* Existem jogos com características anômalas (outliers) no mercado?

---

## Tecnologias Utilizadas
O projeto foi desenvolvido em **Python** utilizando Jupyter Notebook.

* **Pandas & NumPy:** Manipulação e limpeza de dados.
* **Matplotlib & Seaborn:** Visualização de dados e gráficos estatísticos.
* **Scikit-Learn:** Pré-processamento (StandardScaler) e Machine Learning não supervisionado (DBSCAN).
